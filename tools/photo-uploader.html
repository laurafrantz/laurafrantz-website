Setup

This page talks directly to GitHub from your browser — nothing goes through any server but GitHub's own.

  1. Go to github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token.
  2. Repository access: select "Only select repositories" → laurafrantz/laurafrantz-website. Never grant access to all repos.
  3. Permissions: under "Repository permissions", set Contents to Read and write. Leave everything else as "No access".
  4. Set an expiration (e.g. 30 days) and click Generate.
  5. Paste the token into the box on the left and click Connect.

The token is only ever used in this browser tab, to talk directly to api.github.com. It is never sent anywhere else. Closing this page discards it (unless your browser autofills the field later).
